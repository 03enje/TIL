## 폰트 패밀리(font family)

요소의 폰트를 변경할 때 사용합니다.

```css
font-family: Arial;
/* 폰트 스택은 백업용 폰트를 뒤에 나열하여 시스템 상 폰트가 
없을 경우 앞에서부터 순서대로 백업용 폰트를 적용합니다. */
font-family: Big Caslon, sans-serif;
```

항상 제네릭 대체 폰트로 타입을 지정해서 백업을 넣는 게 좋습니다.
